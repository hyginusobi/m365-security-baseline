# Implementation Guide

This folder contains a practical, step-by-step **reference implementation guide** for the Microsoft 365 Security Baseline project.

## Who this is for
- IT Support / EUC / Microsoft 365 administrators
- Security-aware IT teams
- Anyone implementing a baseline aligned with **Cyber Essentials** principles

## Notes
- This guide is written for **lab/reference implementation** and avoids tenant-specific or sensitive information.
- In production environments, follow change control and approval processes.
- Capture evidence safely (redact identifiers, emails, tenant IDs).
